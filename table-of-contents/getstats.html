.getStats()

Auth

Description:

Method .getStats() is used for getting the statistics of rutracker.

Returns:

Promise with response:

{

users: Number,

torrents: Number,

live: Number, // Those torrents that users distribute right now

size: {

value: Number,

measure: String,

},

peer: Number,

seed: Number,

leech: Number,

}

How to use:

rutracker.getStats()

.then((response) => {

// ...

})

.catch((err) => {

// handle errors here

});
