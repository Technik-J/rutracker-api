.getCaptcha()

Description:

Method .getCaptcha() is used for get a captcha. Method takes no arguments.

Returns:

Promise with captcha.

{

img: String, // url to image

capSid: String, // cap_sid field for request

code: String, // code field for request

}

You do not need to use capSid and code fields.

How to use:

rutracker.getCaptcha()

.then(({ img }) => {

// ...

})

.catch((err) => {

// handle errors here

});
