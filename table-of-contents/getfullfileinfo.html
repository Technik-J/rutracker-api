.getFullFileInfo()

Auth

Description:

Method .getFullFileInfo() is used for getting the full info about torrent file (from page with this torrent). Method takes one argument. The id of page.

Arguments:

id: String

Returns:

Promise with response:

{

img: String, // url to poster

seed: String,

leech: String,

hash: String,

body: String, // Full text with HTML tags inside (Too lazy to parse all these things)

categories: [

{

id: String,

name: String,

},

// ...

],

}

How to use:

rutracker.getFullFileInfo('0123456789')

.then((response) => {

// ...

})

.catch((err) => {

// handle errors here

});
