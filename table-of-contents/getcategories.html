.getCategories()

Auth

Description:

Method .getCategories() is used for getting the all categories on Rutracker.org. Method takes one argument - deep.

Arguments:

deep: Boolean

Returns:

Promise with response:

// if 'deep' is false

[

{

id: String,

name: String,

},

// ...

]

// if 'deep' is true

[

{

id: String,

name: String,

subCategories: [

{

id: String,

name: String,

},

// ...

},

// ...

]

How to use:

rutracker.getCategories()

.then((response) => {

// ...

})

.catch((err) => {

// handle errors here

});
