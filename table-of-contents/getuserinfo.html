.getUserInfo()

Description:

Method .getUserInfo() is used for getting the basic information user. Method takes one argument - the id of the user.

Arguments:

id: String

Returns:

Promise with response:

{

username: String,

img: String,

age: String,

role: String,

country: String,

gender: String,

experience: String,

createDate: String, // Temporary the String, will be the Date in the future ...

}

gender can be: 'male', 'female' or 'N/A'

country can be one of these: list of all countries on Rutracker.org

How to use:

rutracker.getUserInfo('0123456789')

.then((response) => {

// ...

})

.catch((err) => {

// handle errors here

});
