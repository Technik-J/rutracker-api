.login()

Description:

Method .login() is used for logging in the user. Method takes 3 arguments. First is the username, second is the password and third is captcha answer.

Arguments:

(username, password, answer)

username: String

password: String

answer: String

Returns:

Promise with cookie.

How to use:

const username = 'username';

const password = 'password';

rutracker.login(username, password, answer) // answer is optional argument

All methods (include .login()) returns the Promise. Inside of promise callback login pass the cookie

.then((cookie) => {

// here we can save our cookie to use them later

})

.catch((err) => {

// handle errors here

});
