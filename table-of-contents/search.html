.search()

Auth

Description:

Method .search() is used for searching for torrents. Method takes 3 arguments. First is the your query for search, second is the sort parameter and third is the direction for sorting.

Arguments:

(query, sortBy, direction)

query: String

sortBy: String

direction: Boolean

sortBy can be any of these: 'size' 'date' 'title' 'seeds' 'leechs' 'downloads'

direction can be: true or false

If direction is true you will get the results sorted from min to max. Vice versa if direction is false

Returns:

Promise with response:

[

{

id: String,

status: String,

title: String,

author: String,

category: {

id: String,

name: String,

},

size: Number,

seeds: Number,

leechs: Number,

downloads: Number,

uploadDate: Date,

url: String,

},

// ...

]

status can be: 'approved' 'doubtfully' 'not-approved' 'temporary'

How to use:

After user logged in you can simply call method .search()

For example, we want to search for 'Star Wars' then sort by size from biggest to smallest.

rutracker.search('Star Wars', 'size', false)

.then((response) => {

// ...

})

.catch((err) => {

// handle errors here

});
